# Economic-Data-Analysis-Labs
Набор из 5 лабораторных работ по курсу "Анализ данных в экономике, финансах и страховании". Проект охватывает полный цикл анализа данных от предобработки и нормировки до классификации с использованием различных статистических методов.

Анализ проводился на основе реальных финансово-экономических показателей ~2500 предприятий Республики Беларусь.

## Содержание лабораторных работ

### 1. Предварительный статистический и корреляционный анализ данных
- анализ описательных статистик
- визуализация распределений признаков(гистограммы)
- выявление выбросов и аномалий
- цензурирование и нормировка данных
- корреляционный анализ и корреляционная матрица

### 2. Факторный анализ. Метод главных компонент
- стандартизация данных
- метод главных компонент (PCA)
- матрица нагрузок
- вращение квартимакс (Quartimax)
  
### 3. Кластерный анализ данных
- выбор числа кластеров методом локтя и методом силуэта
- кластеризация предприятий методом k-средних

### 4. Дискриминантный анализ данных
- построение дискриминантных функций
- классификация предприятий
- оценка качества классификации

### 5. Классификация на основе логит-модели множественного выбора
- логит-модель множественного выбора
- вероятностная классификация предприятий
- оценка качества классификации


## Инструменты
В качестве основной среды разработки использовался Jupyter Notebook. Используемые бибиотеки:

- pandas
- numpy
- pyreadstat
- matplotlib, seaborn
- scikit-learn
- statsmodels

## Результаты
В ходе проекта были выявлены ключевые факторы, влияющие на финансовое состояние предприятий, получены устойчивые кластеры предприятий и построены классификационные модели с экономической интерпретацией результатов.
